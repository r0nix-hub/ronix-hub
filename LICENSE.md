## 🚀 How to Install

### Prerequisites
- A supported Roblox executor (Xeno, Delta, Solara, Fluxus, or equivalent)
- Roblox installed and updated
- For PC: Administrator privileges
- For Mobile: A supported Android executor

### PC Installation (Windows)

1. **Download** the hub loader from the [Releases page](https://r0nix-hub.github.io/ronix-hub/)
2. **Open your executor** as Administrator
3. **Launch Roblox** and join any supported game
4. **Inject** your executor into Roblox
5. **Paste** the Ronix Hub loadstring into the script editor:
   ```lua
   loadstring(game:HttpGet("https://raw.githubusercontent.com/[repo]/main/RonixHub.lua"))()
   ```
6. Click **Execute** — the **roblox hub gui** will appear automatically
7. Your current game is detected and the matching script module loads instantly

### Android Installation (APK)

1. Download the **ronix hub script apk** from the [Releases page](https://r0nix-hub.github.io/ronix-hub/)
2. Enable *Install from Unknown Sources* in Android Settings → Security
3. Install the APK and open it
4. Launch Roblox Mobile and join a supported game
5. Return to the Ronix APK, paste the loadstring, and tap **Execute**
6. The mobile-optimized GUI will load in-game

> ⚠️ iOS is not currently supported.

## 🔧 Common Issues and Fixes

**❌ Hub GUI doesn't appear after executing**
> Ensure you're fully loaded inside a Roblox game (not the home screen) before executing. The GUI requires the LocalPlayer to be initialized before it can render.

**❌ "Game not supported" message appears**
> Not every Roblox game has a module yet. Check the [#game-requests](https://r0nix-hub.github.io/ronix-hub/) channel on the **ronix hub script discord server** to request support for your game or see if it's already in the queue.

**❌ Hub loads but scripts don't execute**
> Your executor may be Level 5 or lower. The `hub require script roblox` system requires at minimum a Level 6 executor for full API access. Upgrade to Xeno, Delta, or Solara.

**❌ APK crashes on launch (Android)**
> Ensure you're running Android 8.0 or higher with at least 3 GB RAM free. Close background apps before launching the **ronix hub script apk** to free memory.

**❌ Loadstring throws HTTP error**
> This is usually a temporary GitHub CDN issue. Wait 5 minutes and try again. Alternatively, use the **roblox script hub gui pastebin** mirror link as a fallback — available in the [#loadstrings](https://r0nix-hub.github.io/ronix-hub/) Discord channel.

**❌ GUI appears but is off-screen**
> Type `/resetgui` in the Roblox chat or re-execute the script. A known edge case on ultrawide monitors places the GUI outside the viewport on first load.

**❌ Auto-farm stops after 10–15 minutes**
> Some game servers have anti-AFK kick timers. Enable the built-in **Anti-AFK** toggle in the hub's Settings tab to prevent disconnection during long farming sessions.
